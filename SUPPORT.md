## 💼 Monthly Backers

Sustained monthly support helps us maintain infrastructure, pay for CI/CD, and fund core development sprints.

### GitHub Sponsors

The easiest way to support us monthly:

[**Become a GitHub Sponsor**](https://github.com/sponsors/ed2kIA)

Tiers:
- **Individual Backer** ($5/mo): Name in backers list + early access to releases
- **Supporter** ($25/mo): Discord role + monthly dev updates
- **Organizational Backer** ($100/mo): Logo on README + direct input on roadmap priorities
- **Enterprise Partner** ($500+/mo): Custom SLA + priority support

## ❓ FAQ

**Q: Can I donate hardware?**
A: Yes! GPU compute, servers, or networking equipment. Contact us at `infra@ed2kIA.org`.

**Q: How are funds managed?**
A: Via 2-of-3 multisig wallet + Open Collective for transparent expense tracking.

**Q: Can I propose fund allocation?**
A: Yes! Any contributor with 100+ reputation points can submit funding proposals for community vote.

**Q: Is there tax deduction?**
A: Currently ed2kIA is a decentralized project. We're exploring 501(c)(3) foundation structure for tax-deductible donations.
