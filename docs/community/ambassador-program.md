## Overview

The ed2kIA Ambassador Program empowers community leaders to drive adoption, education, and growth of the ed2kIA decentralized AI network. Ambassadors represent ed2kIA at events, create educational content, mentor new contributors, and provide feedback from their communities.

---

## Ambassador Tiers

### 🌱 Seed Ambassador

**Requirements:**
- Active GitHub account with ≥ 5 contributions to open source
- Basic understanding of blockchain/AI concepts
- Commitment: 2-4 hours/week

**Responsibilities:**
- Share ed2kIA content on social media (≥ 2 posts/week)
- Participate in community Discord/Forum discussions
- Report bugs and suggest improvements
- Attend monthly community calls

**Benefits:**
- Ambassador badge on GitHub/Discord
- Early access to new features
- Direct line to core team
- Quarterly swag package

### 🌿 Sprout Ambassador

**Requirements:**
- Seed Ambassador for ≥ 3 months
- Organized ≥ 1 community event (meetup, workshop, talk)
- Created ≥ 3 educational resources (blog posts, tutorials, videos)
- Commitment: 4-8 hours/week

**Responsibilities:**
- Organize monthly community events
- Mentor Seed Ambassadors
- Create technical content (tutorials, guides, demos)
- Provide community feedback to core team
- Help onboard new contributors

**Benefits:**
- All Seed benefits +
- Travel stipend for conferences ($500/event)
- Featured in ed2kIA newsletter
- Advisory input on roadmap priorities
- Grant co-author opportunities

### 🌳 Tree Ambassador

**Requirements:**
- Sprout Ambassador for ≥ 6 months
- Built active community chapter (≥ 50 members)
- Led ≥ 3 successful events
- Contributed code/docs to ed2kIA repository
- Commitment: 8-12 hours/week

**Responsabilities:**
- Lead regional community chapter
- Speak at conferences/hackathons
- Contribute to core development or documentation
- Recruit and mentor new ambassadors
- Represent ed2kIA in media/interviews

**Benefits:**
- All Sprout benefits +
- Conference travel + speaking stipend ($2000/event)
- Core team collaboration opportunities
- Revenue share from community-driven grants
- Lifetime ambassador status

## Onboarding Checklist

### For New Seed Ambassadors

- [ ] Complete application form
- [ ] Join ed2kIA Discord #ambassadors channel
- [ ] Read ed2kIA documentation (README, GOVERNANCE, CONTRIBUTING)
- [ ] Set up development environment (follow quickstart guide)
- [ ] Introduce yourself in #ambassadors channel
- [ ] Schedule 1:1 onboarding call with community lead
- [ ] Create first social media post about ed2kIA
- [ ] Join next community call as participant

### For Sprout Promotion

- [ ] Document first community event
- [ ] Publish first educational resource
- [ ] Mentor at least 1 Seed Ambassador
- [ ] Submit promotion request to community lead
- [ ] Review and accept Sprout responsibilities

### For Tree Promotion

- [ ] Build active chapter (≥ 50 members)
- [ ] Lead ≥ 3 successful events
- [ ] Contribute to core repository
- [ ] Submit promotion request with portfolio
- [ ] Core team review and approval
