## 3. Achievement Badges

### 3.1 Badge Format

Badges are generated as SVG with Ed25519 proof placeholder:

```
ed2kIA-badge-{TIER}-{ACHIEVEMENT}-{TIMESTAMP}
Proof: {Ed25519_SIGNATURE_PLACEHOLDER}
```

### 3.2 Badge Categories

| Category | Color | Prefix |
|----------|-------|--------|
| Technical | Blue | `tech-` |
| Community | Green | `comm-` |
| Sustained | Gold | `time-` |
| Security | Red | `sec-` |
| Special | Purple | `spec-` |

### 3.3 Badge Display

Badges can be displayed in:
- GitHub profile README
- Pull request comments
- Contributor pages
- Release notes
- Community forum profiles

## 8. Integration with Governance

### 8.1 Voting Rights

Contributors with Gold tier or above receive voting rights in:
1. Feature prioritization
2. RFC approval
3. Maintainer elections
4. Governance amendments

### 8.2 Maintainer Path

Path to maintainer:
1. Achieve Platinum tier
2. Nominated by existing maintainer
3. Community vote (simple majority)
4. 3-month probation period
5. Full maintainer status
