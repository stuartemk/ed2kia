## Overview

## 5. New Modules Quick Start

### 5.1 Proof Aggregation

```rust
use ed2kia::zkp::proof_aggregation::{ProofAggregator, AggregationBatch};

let mut aggregator = ProofAggregator::new(100, 10);
let batch_id = aggregator.create_batch(1000)?;
aggregator.add_proof_to_batch(&batch_id, "proof_1")?;
aggregator.add_proof_to_batch(&batch_id, "proof_2")?;
aggregator.finalize_batch(&batch_id, 1100)?;
assert!(aggregator.verify_batch(&batch_id)?);
```

### 5.2 Neural Steer UI

```rust
use ed2kia::gui::neural_steer_ui::{NeuralSteerConfig, SteeringSignalBridge};

let config = NeuralSteerConfig::new(0.7, 0.5, 0.9, 1000);
assert!(config.is_safe());

let mut bridge = SteeringSignalBridge::new("ui_source".to_string(), 1000);
bridge.apply_config(config.clone())?;
```

## 7. Troubleshooting

### Q: Build fails with "feature not found"

**A:** Ensure you're using the correct feature flags:
```bash
cargo build --release --features "v1.9-sprint1,v1.9-sprint2"
```

### Q: Tests fail after migration

**A:** Run with feature flags enabled:
```bash
cargo test --features "v1.9-sprint1,v1.9-sprint2"
```

### Q: Clippy warnings appear

**A:** 3 style warnings are expected and non-blocking. Suppress with:
```bash
cargo clippy --features "v1.9-sprint1,v1.9-sprint2" -- -W clippy::all -A clippy::style
```

### Q: WASM sandbox errors

**A:** Verify wasmtime configuration:
- Backend: Cranelift (default)
- Memory limit: 256MB
- WASI: Minimal (no filesystem/network)
