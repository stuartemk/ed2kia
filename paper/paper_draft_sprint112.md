Neural ODE Zonotope Reachability\n\n### 2.1 Mathematical Foundation\n\nA Neural ODE models continuous-time dynamics:\n\n```\ndx/dt = f_θ(x(t), t)\n```\n\nwhere `f_θ` is a neural network parameterized by `θ`. The solution at time `T` is:\n\n```\nx(T) = x(0) + ∫₀ᵀ f_θ(x(t), t) dt\n```\n\n### 2.2 Flowpipe Computation\n\nA flowpipe is a sequence of zonotopes `Z(t₀), Z(t₁), ..., Z(t_T)` that over-approximates the reachable set at each time step:\n\n```\nZ(t_{k+1}) = Φ(Z(t_k))\n```\n\nwhere `Φ` is the integration step (Euler, RK2, or RK4).\n\n**Euler Step**: `Z(t+dt) = Z(t) + dt · f_θ(Z(t))`\n\n**RK2 (Heun)**: Two-stage predictor-corrector with midpoint evaluation.\n\n**RK4 (Classical)**: Four-stage Runge-Kutta with weights `[1/6, 2/6, 2/6, 1/6]`.\n\nEach step propagates the zonotope through the neural field `f_θ`, applying affine transformation followed by activation approximation (ReLU, SiLU, GELU).\n\n### 2.3 Integration Methods\n\n| Method | Stages | Order | Stability | Use Case |\n|--------|--------|-------|-----------|----------|\n| Euler | 1 | O(dt) | Conditional | Fast exploration |\n| RK2 | 2 | O(dt²) | Better | Balanced |\n| RK4 | 4 | O(dt⁴) | Best | Precision |\n\n### 2.4 Volume Growth and Stability\n\nThe flowpipe volume proxy tracks over-approximation quality:\n\n```\nlog_vol(Z(t)) = Σ_j log(width_j(t))\n```\n\nMonotonic volume growth indicates stable dynamics. Sudden volume explosion suggests instability or excessive wrapping effect.\n\n### 2.5 Trajectory Certificate\n\nA `TrajectoryCertificate` summarizes the flowpipe analysis:\n\n```\ncert = {\n    epsilon: min_j(width_j) / 2,           // Safety radius\n    violation_prob: MC_violations / N,     // Monte Carlo violation rate\n    num_steps: flowpipe.steps.len(),        // Total time steps (includes initial)\n    total_time: num_steps * dt,             // Time horizon\n}\n```\n\nThe certificate is valid when `violation_prob < 1/N` (confidence `1 - 1/N`).\n\n---\n\n## 3. Control Barrier Functions (CBFs)\n\n### 3.1 Linear CBF\n\nA Control Barrier Function `h: R^n → R` defines a safe set `C = {x | h(x) ≥ 0}`. Our linear CBF:\n\n```\nh(x) = w^T · x + b\n```\n\nwhere `w ∈ R^n` (weight vector) and `b ∈ R` (bias).\n\n### 3.2 Lie Derivative\n\nThe Lie derivative along the vector field `f_θ`:\n\n```\nL_f h(x) = ∇h(x) · f_θ(x) = w^T · f_θ(x)\n```\n\nFor forward invariance of `C`, we require:\n\n```\nL_f h(x) + α·h(x) ≥ 0\n```\n\nwhere `α > 0` is the decay rate parameter.\n\n### 3.3 Zonotope CBF Evaluation\n\nFor zonotope `Z = (c, G)`, the lower bound of `h(x)`:\n\n```\nh_min(Z) = w^T · c + b - Σ_i |w^T · g_i|\n```\n\nIf `h_min(Z) ≥ 0`, the entire zonotope is certified safe.\n\n### 3.4 Norm-Based CBF\n\nFor spherical safety regions:\n\n```\nh(x) = R² - ||x||²\n```\n\nwhere `R` is the safety radius. Select winners to maximize social welfare: `W* = argmax_W Σ_{i∈W} v_i`\n2. Each winner pays the externality: `p_i = Σ_{j∉W*} v_j(W*\\{i}) - Σ_{j∉W*} v_j(opt\\i)`\n3. Formal Properties\n\n### 7.1 Theorem: Flowpipe Soundness\n\n**Statement**: If `Z(0)` contains `x(0)` and `Φ` is a sound integration step, then the flowpipe `Z(0), ..., Z(T)` contains the true trajectory `x(0), ..., x(T)`.\n\n**Proof Sketch**: By induction on time steps. Inductive step: If `x(t) ∈ Z(t)`, then `x(t+dt) = x(t) + dt·f_θ(x(t)) ∈ Φ(Z(t)) = Z(t+dt)` by soundness of `Φ`.\n\n### 7.2 Theorem: CBF Forward Invariance\n\n**Statement**: If `h_min(Z(t)) ≥ 0` for all `t`, and `L_f h(x) + α·h(x) ≥ 0` for all `x ∈ Z(t)`, then the trajectory remains in the safe set `C = {x | h(x) ≥ 0}`.\n\n**Proof Sketch**: By the comparison lemma. The condition `L_f h + α·h ≥ 0` implies `d/dt(e^{αt}·h(x(t))) ≥ 0`, hence `h(x(t)) ≥ e^{-αt}·h(x(0)) ≥ 0`.\n\n### 7.3 Theorem: VCG Truthfulness\n\n**Statement**: In a VCG auction, reporting true value `v_i` is a dominant strategy for each bidder `i`.\n\n**Proof Sketch**: The VCG payment `p_i` depends only on other bidders' values, not on `i`'s bid.
