📐 Filosofía del Kernel Estuardiano

Eje Z (Alineación Ética): Tensores con Z > 0 son atraídos al Foco Superior. Z < 0 caen al Foco Inferior.

Crédito de Existencia (CE): Emisión por compute ético (Z > 0). Quema por perversidad (Z < 0).

Apoptosis: Cuando CE ≤ -100, el Sistema Inmunológico recicla el nodo aberrante. Healthy → Pain → Apoptosis.
